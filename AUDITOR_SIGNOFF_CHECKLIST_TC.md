## 2) 系統已覆蓋邏輯（工具驗證）

請對照系統畫面及匯出工作底稿核對後勾選：

- [ ] 轉制分界日為 `2025-05-01`。
- [ ] 年資採用「足年 + 餘下天數 / 365」。
- [ ] 當評估日早於轉制日，轉制前年資已以評估日封頂。
- [ ] 已套用 LSP 薪金上限（`HK$22,500/月`）及總上限（`HK$390,000`）。
- [ ] 強制性 MPF 僅可對沖轉制前 LSP。
- [ ] 自願性 MPF 先對沖轉制前，再以餘額對沖轉制後。
- [ ] 政府資助政策年度按每年 5 月 1 日週年切換。
- [ ] 補貼門檻分流（`<=500,000` 與 `>500,000`）套用正確。
- [ ] HKFRS/HKFRS for PE 路徑已套用折現因子及留任概率。
- [ ] SME-FRS 路徑不折現（`PV=1`）且不作流失調整（`prob_stay=1`）。

證據文件：
- `REGULATION_TRACEABILITY.md`
- 匯出工作底稿：`2.Assumptions`、`4.Calc Detail`、`5.Limitations`

## 3) 數據與來源證據（需人工核對）

- [ ] 員工主數據與人事記錄一致（姓名、入職日、出生日期）。
- [ ] `Salary at Transition` 為 `2025-04-30` 當日正確月薪。
- [ ] `Current Salary` 為評估日正確月薪。
- [ ] MPF 結餘為受託人於評估日確認之金額。
- [ ] 所有估算 MPF 強制性結餘在最終簽核前已由實際結單替換。
- [ ] 特殊個案（如 2000 年前入職、特殊合約條款）已另行覆核。

主要證據文件：
- HR / Payroll 報表
- MPF 受託人結單
- 本次工具輸入檔（CSV / XLSX）
